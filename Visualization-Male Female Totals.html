Male vs Female Total Applications



The number of applications slightly declined incrementally from 1960 to 1970, with a slight increase in applications in 1970. Each year, the number of male applications seems to be at the same proportionate increase above the female applications, with the lowest number of applications for both males and females stemming from 1967 and 1968.
