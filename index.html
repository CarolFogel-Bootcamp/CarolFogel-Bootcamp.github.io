Social Security Card Applications (1960 - 1970)


The purpose of this project is to create a responsive visualization dashboard website using HTML, CSS, and Bootstrap. Graphics are used to tell a story. This project tells the story of Social Security Card Applications received. The data was retrieved from Data.Gov (https://catalog.data.gov/dataset/baby-names-from-social-security-card-applications-national-level-data) within 'Baby Names from Social Security Card Applications - National Level Data'. The Data.Gov website has Social Security card application data spanning from 1880 to 2018.

This website provides information on applications received from 1960 to 1970. Analysis has uncovered the most popular female names and male names during this time period. In addition, analysis was done on the number of applications received during this time period, as well as the breakdown of names by first initial to determine which letter is the first letter of the majority of names during this time period.
